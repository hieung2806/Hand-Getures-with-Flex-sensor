# Hand-Getures-with-Flex-sensor

With Flex sensor and accelerometer MPU6050, building models to regconize Vietnamese sign language with 7 class : "Vâng","Ăn","Uống", "Dừng", "Không", "Đưa", "Không có cử chỉ".
